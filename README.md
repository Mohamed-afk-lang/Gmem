# AI Knowledge Quest Web Version

Open `index.html` in a browser to play the game as a website.

Alternatively, run `serve.bat` or `serve.ps1` to launch a local static server and open the page at `http://localhost:8000`.

This is a static web version of the AI Knowledge Quest quiz and inference demo.

## Files

- `index.html` — page structure and app shell
- `styles.css` — page styling
- `script.js` — quiz logic and game flow

## Deploy to a shareable link with GitHub Pages

1. Create a GitHub repository and add this project.
2. Commit all files and push to the `main` branch.
3. Enable GitHub Pages and set the source branch to `gh-pages` if required.
4. The workflow at `.github/workflows/pages.yml` will publish the site automatically after each push.

Once deployed, GitHub Pages will provide a public URL that anyone can open.
